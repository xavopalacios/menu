⚔️ Elementos Clave para el "Look" Pro
Tipografía: Busca una fuente llamada "Radiance" (es la oficial de Dota 2) o usa "Cinzel" de Google Fonts para ese toque épico.
Hover Effects: En Dota, nada es estático. Agrega un transition suave. Cuando el usuario pase el mouse, el brillo debe aumentar gradualmente.
Overlay de Textura: Agrega una capa semitransparente con ruido o textura de piedra sobre el fondo (opacity: 0.1) para quitar la apariencia plana de "página web común".

.btn-dota {

    content: '';
    position: absolute;
    top: 0;

    height: 100%;
    background: linear-gradient(
        to right,
        transparent,
        rgba(255, 255, 255, 0.2),
        transparent
    );
    transform: skewX(-25deg);
    transition: 0s;
}

/* Animación al pasar el mouse */
